ng-swiper
=========

use swiper with angular and requirejs

usage:
`
<swiper current-slide-index="0" show-pager="false" only-external="true" animation-name="reverse">
    <swiper-slide class="main">
        <scroll mode="vertical">
            <ul>
                <li></li>
                <li></li>
                <li></li>
                <li></li>
            </ul>
        </scroll>
    </swiper-slide>
</swiper>
`
